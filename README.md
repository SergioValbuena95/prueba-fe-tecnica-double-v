# Prueba Frontend Developer

This project is a frontend application.

## Installation

1. Clone the repository: `git clone https://github.com/your-username/pruebafront.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Commands

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the production-ready code.
- `npm run preview`: Starts the production server.

## Architecture

The project follows a modular architecture with the following components:

- [src/components]: Contains the reusable components of the application.
- [src/pages]: Contains the pages of the application.
- [src/store]: Contains the Pinia store for managing the application state.
- [src/assets]: Contains the static assets of the application.

The project uses the following libraries:

- Vue.js: A progressive JavaScript framework for building user interfaces.
- Vite: A build tool for modern web development.
- Vue Router: A routing library for Vue.js applications.
- Pinia: A state management library for Vue.js.
- Axios: A promise-based HTTP client for making API requests.
- ECharts: A charting library for data visualization.
